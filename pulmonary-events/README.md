# Pulmonary Medicine Scientific Events

Static landing page for Pulmonary Medicine scientific events.

## Files
- `index.html`: public landing page.
- Brochure images are embedded in the HTML for previous events.

## Update format for future events
Send the following data to update the `events` array in `index.html`:

```text
status: Upcoming
titleAr:
titleEn:
medicalField:
eventType:
date:
time:
location:
moderator:
organizers:
speakers:
agenda:
brochureImage: public image URL or embedded data URL
brochureLink:
registrationLink:
displayOnPublicPage: true
```

## GitHub Pages
If GitHub Pages is enabled for this repository, the expected project URL is:

`https://cpdaatc.github.io/cpdaatc/pulmonary-events/`

If it is not active, enable GitHub Pages from repository Settings → Pages → Deploy from branch → main → /root.
